# ☕ Cafe Management System (MERN)

🔗 **Live Website:** https://cafe-management-frontend.onrender.com/

## 📌 About the Project

The **Cafe Management System** is a full-stack web application built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.
This is my **first full-stack project**, created to understand how frontend, backend, and database work together in a real-world application.

The system allows an **admin to manage cafe operations**, including menu items and orders through a clean and responsive interface built with **React and Tailwind CSS**.

## 🚀 Features

* 🔐 **Admin Authentication**

  * Only **admin login** is required to access and manage the system.

* 🍔 **Menu Management**

  * Add new food and drink items
  * Update existing menu items
  * Delete items from the menu

* 🧾 **Order Management**

  * Create and manage customer orders
  * Track ordered items

* 📊 **Admin Dashboard**

  * Manage cafe data from a centralized panel

* 🎨 **Modern UI**

  * Responsive and clean interface using **Tailwind CSS**
